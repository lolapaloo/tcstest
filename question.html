Create a Movie Finder app which should list all available movies and will have a TextInput to accept user input on which movies will be searched.

 

  • Movie List should list   Title,Actors and image of the Movie.
  • The user can input both Title or any Actor to search for a movie.
  • Search should be case insensitive.
  • Movielist should render searched movies as user types in TextInput.
  • movie.js contains information about books.

Note :

  • If you get "port already in use" error while running the application, kill the application and re-run the rest.
  • Since React-Native will be running on the browser, there are some limitations on component support, however, the question is designed keeping in mind those limitations.
  • Install all dependencies before testing.
  • To install dependencies, run->install.
  • to test, run->test.
